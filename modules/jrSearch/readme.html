Jamroom5 jrSearch Module

Revision History
  • V1.0.0 dd mmm yyyy - Initial release.
Description
  • Provides a site datastore search feature
  • Smarty template calls for a search form, recent and popular searches
Installation
  • Copy the module folder to your /modules folder.
  • Login as admin and this module should show under the modules Media category.
  • It shouldn't be necessary to run the Integrity Check, but if there are any issues, do it anyway.
Module Configuration
  • Click on the module's 'Global Config' tab. A page is displayed where you can select whether to allow search for logged in users only.
Template Integration
  • Smarty call {jrSearch_form} to return a search form
  • Parameters are -
    • module: module name, comma separated module names, or 'all' to search all modules - default: all
    • page: optional - default 1
    • pagebreak: optional - default 10
    • value: optional - default site search (language file 1)
    • template: optional - module's html_search_form.tpl
    • class: optional - passed through to the template
    • style: optional - passed through to the template
  • Smarty call {jrSearch_recent} to return recent search strings
  • Parameters are -
    • user_id: optional - return the specified user's recent searches
    • module: optional - return the specified module's recent searches
    • limit: optional - default 5
    • template: optional - module's search_recent.tpl
    • class: optional - passed through to the template
    • style: optional - passed through to the template
  • Smarty call {jrSearch_popular} to return popular search strings
  • Parameters are -
    • limit: optional - default 5
    • template: optional - module's search_popular.tpl
    • class: optional - passed through to the template
    • style: optional - passed through to the template
